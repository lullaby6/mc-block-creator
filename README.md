# MC Block Creator

> Custom Block with Display Entities Generator for Minecraft Java Edition

https://mc-block-creator.vercel.app/

## Future features

-   Tools draggable card
-   Layers draggable card
-   Layers draggable card
-   Stretchable pixels
-   Layout options for Solid, Tables and Unique blocks
-   Pallete block select input
-   Base block select input
-   Support block_display and text_display
-   PWA

## Contributing

This project uses [React](https://es.react.dev/) and [Vite](https://vitejs.dev/).

1. Make sure you have [node](https://nodejs.org/), npm, and [git](https://git-scm.com/) installed.
2. Clone the repository.

```bash
git clone https://github.com/lullaby6/mc-block-creator.git
```

3. Start the dev server.

```bash
npm run dev
```

4. Open the browser in `localhost:3000`.
